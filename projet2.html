du projet

Tous les sites qui ont une base d’utilisateurs ont besoin de pages permettant de créer un compte, se connecter et redéfinir son mot de passe. Vous devez donc créer ces trois pages : la première contient un formulaire avec login, mot de passe et un bouton de validation En bas de ce formulaire se trouve un lien qui renvoie vers la page de création de compte et la redéfinition de mot de passe.

La deuxième page doit avoir un formulaire complet : login (obligatoire), mot de passe et confirmation du mot de passe (obligatoire), date de naissance formatée, nom, prénom, adresse email (obligatoire et correctement formatée), sexe (obligatoire, choix Homme/Femme), niveau d’étude (obligatoire, dans une liste), numéro de téléphone, une présentation sur plusieurs lignes, un choix multiple de hobbies et un bouton de validation qui ramène sur la page de connexion. Tous les champs doivent avoir des labels. Enfin sur la page de mot de passe oublié, il suffit de renseigner son adresse email. Toutes les pages doivent utiliser le même css.

De beaux formulaires avec des inputs personnalisés sont toujours plus agréables pour l’utilisateur !

Page connexion

page principal pour se connecter avec des inputs fluid et qui change la couleur quand ils sont actives, avec deux liens en bas à gauche pour reinisialiser le mot de passe en cas d'oublie et a droite pour s'inscrire

Page Mot de pass oublié

deuxieme page html avec le meme CSS avec un seul input de type E-mail pour reinisialiserle mot de passe, et en pas deux liens pour se deriger vers la page de connexon et le deuxieme pour la page d'inscription

Page inscription

la derniere page est la page d'inscription qui comporte plusieurs inputs et une liste déroulante un choix de sexe et d'autre inputs plus un textarea tous en gardant toujours le meme CSS.
